# The-Frost-Home - Дипломный проект

**Разработка 3D-игры в жанре Survival с системой крафта и движущимися погодными условиями**

---

## 🎮 Об игре

The Frost Home — это 3D-экшен в жанре выживания, выполненный в сеттинге вечной мерзлоты, где игроку предстоит бороться за жизнь в суровом мире. Основные механики:

- Исследование мира
- Сбор ресурсов
- Создание инструментов и оружия
- Управление температурой, голодом и здоровьем
- Динамическая погода (ветер, снег, мороз)

---

## 🛠️ Используемые технологии

- **Движок:** Unity 2022.3.42f1
- **Язык программирования:** C#
- **Графика:** 3D
- **Система контроля версий:** Git (GitHub)

---

## 📦 Содержание проекта

- `Scripts/` — скрипты игры
- `Models/` — 3D-модели
- `Textures/` — текстуры
- `Prefabs/` — префабы
- `Scenes/` — сцены
- `Audio/` — звуковые файлы

---

## 🎯 Основные механики

- **Инвентарь** — система хранения предметов
- **Крафт** — создание предметов из ресурсов
- **Погода** — влияет на температуру и выживание
- **Третье лицо** — управление персонажем с видом от третьего лица

---

## 🕹 Управление

- WASD - управление
- Space - прыжок
- I - инвентарь
